## [0.0.0] - 2026 - 05 - 07

### Added
- Initial release of GlyComboCLI
- Support for native, permethylated, and peracetylated derivatisations
- Support for Free, Reduced, InstantPC, Rapifluor-MS, 2-AA, 2-AB, Procainamide, Girard's reagent P, and custom reducing ends
- mzML input support for Bruker, Thermo, Agilent, Waters, and Sciex instruments
- Plain text, mzML and Galaxy mass list input support
- Positive and negative mode adducts: MH+, MNa+, MNH4+, MK+, MH-, MFA-, MAA-, MTFA-, Neutral
- Modified sialic acid support: lactonised, ethyl esterified, dimethylamidated, and ammonia amidated NeuAc and NeuGc
- Phosphate, sulfate, dHexNAc, and acetylation modifications
- Up to five user-defined custom monosaccharides
- Custom reducing end support
- Custom adduct support
- Skyline-compatible CSV output for mzML input
- Parameter log file output
- Off-by-one isotope correction flag
- ppm and Da mass error modes
